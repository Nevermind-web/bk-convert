# BK Convert

BK Convert je moderní webová aplikace pro konverzi obrázků a video souborů přímo v prohlížeči.  
Aplikace funguje bez backendu, bez instalace a je plně kompatibilní s GitHub Pages.

---

## Funkce

### Obrázky
- PNG, JPG, WEBP, BMP
- Konverze mezi formáty
- Hromadná (batch) konverze
- Funguje offline i online

### Videa
- MP4, WEBM, AVI, MOV, MKV
- Konverze nahraných video souborů
- Běží pomocí FFmpeg.wasm
- Vyžaduje HTTPS (GitHub Pages)

### Uživatelské rozhraní
- Fullscreen moderní design
- Animace a mikrointerakce
- Drag & Drop podpora
- Tmavý režim
- Přehledné přepínání režimů
